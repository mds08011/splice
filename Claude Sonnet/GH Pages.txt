GitHub Pages (recommended — free, permanent)

Create a free account at github.com
Click + → New repository → name it meeting-recorder → set to Public → Create
On the repository page, click Add file → Upload files → drag in the downloaded HTML file
Before committing, rename it to index.html in the filename box
Click Commit changes
Go to Settings → Pages → under Source, select Deploy from branch → branch: main, folder: / (root) → Save
Wait about a minute — your URL appears at the top of that same Settings → Pages page: https://[yourusername].github.io/meeting-recorder

Bookmark that URL on your phone and it works as a web app from any browser, any device. When you want to update the tool later, just replace index.html in the repo and the live URL updates automatically.
